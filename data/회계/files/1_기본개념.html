  • 기본 개념
    1. 손익계산서
      2. : 손실과 이익을 정리하고 기록한 재무제표
         - 결산하고 나면 다음 해 초부터는 다시 0부터 출발
    2. 재무상태표
      3. : 재산의 변화를 기록한 재무제표
         - 해마다 누적해서 기록
    3. 영업외수익
      4. : 기업의 주된 경영 활동이 아닌 다른 활동으로 벌어들인 수익
    4. 영업외비용
      5. : 기업의 주된 경영 활동이 아닌 다른 활동이 원인이 되어 발생한 비용
  • Tip
    1. 회계항등식 : 자산 = 자본(자본금 + 이익잉여금) + 부채
    2. 이익 = 수익 - 비용
    3. 매출총이익 = 매출 - 매출원가
    4. 영업이익 = 매출총이익 - 판매비 및 관리비
    5. 법인세차감전이익(세전이익) = 영업이익 + 영업외수익 - 영업외비용
    6. 당기순이익 = 법인세차감전이익(세전이익) - 법인세비용
