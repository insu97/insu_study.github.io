  • Database의 정의

    •     : Database는 특정 조직의 여러 사용자가 '공유'하여 사용할 수 있도록 '통합'해서 '저장'한 '운영' 데이터의 집합

  • Database의 특징
    1. 데이터베이스는 실시간 접근이 가능하다.
    2. 데이터베이스는 계속 변화한다.
    3. 데이터베이스는 동시 공유가 가능하다.
    4. 데이터베이스는 내용으로 참조가 가능하다.
  • 데이터 종류

    1. 정형 데이터
      2.  : 정형 데이터는 구조화된 데이터 , 즉 미리 정해진 구조에 따라 저장된 데이터.
        Ex) Excel , CSV , ...

    2. 반정형 데이터
      3.  : 반정형 데이터는 구조에 따라 저장된 데이터이지만 정형 데이터와 달리 데이터 내용 안에 구조에 대한 설명이 함께 존재한다.
        Ex) HTML , XML , JSON ...

    3. 비정형 데이터
      4.  : 비정형 데이터는 정해진 구조가 없이 저장된 데이터다.
        Ex) 텍스트 , 영상 , 이미지 , 워드 , PDF ...
